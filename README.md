# Natural Language Generator

An AI-powered Natural Language Generator web application built using Flask and OpenRouter API.

## Features

- Generate human-like text using AI
- Clean and modern user interface
- Flask backend integration
- OpenRouter API integration
- Responsive design
- Environment variable security using `.env`

## Technologies Used

- Python
- Flask
- HTML
- CSS
- OpenRouter API
- OpenAI SDK

## Installation

### Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

### Move Into Project Folder

```bash
cd natural-language-generator
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Configure Environment Variables

Create a `.env` file and add:

```env
OPENROUTER_API_KEY=your_api_key_here
```

## Run Application

```bash
python app.py
```

Open browser:

```plaintext
http://127.0.0.1:5000
```
